Escolhas de Privacidade

Esta página explica as opções disponíveis para controlar, apagar ou solicitar esclarecimentos sobre os dados usados no Simulado.

1. Dados armazenados localmente

O Simulado foi desenvolvido para funcionar sem criação de conta. Os registros de simulados, desempenho, observações e humor pós-prova ficam armazenados localmente no dispositivo do usuário.

2. Como apagar registros

Quando houver opção disponível no app, o usuário poderá apagar registros diretamente na interface do aplicativo. Também é possível remover os dados locais ao desinstalar o app do dispositivo, conforme o comportamento do iOS.

3. Como parar de usar o app

O usuário pode parar de usar o Simulado a qualquer momento. Como o app não exige conta, não há assinatura, login ou perfil remoto a ser encerrado.

4. Publicidade e rastreamento

O Simulado não utiliza os dados registrados para publicidade comportamental e não vende dados pessoais.

5. Solicitações e esclarecimentos

Para dúvidas sobre privacidade, uso dos dados ou funcionamento do app, entre em contato pelo e-mail: suporte@aldim.app

6. Política completa

Para mais detalhes, consulte a Política de Privacidade do Simulado.
